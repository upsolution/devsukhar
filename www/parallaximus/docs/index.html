Parallaximus Documentation

Basic usage

1. Upload parallaximus folder to your server.

2. Include basic Parallaximus styles before your main page styles:

<!-- Basic Parallaximus styles  -->
<!-- (should be inserted before specific widget styles to be overloaded) -->
<link rel="stylesheet" type="text/css" href="parallaximus/parallaximus.css" media="all">

3a. For jQuery version (click for MooTools) include this scripts:

<!-- Parallaximus requires JQuery 1.7+ -->
<!-- Note: if you already have jQuery at this moment, you shouldn't include it once more. -->
<script type="text/javascript" src="parallaximus/jquery-1.9.1.min.js"></script>

<!-- Parallaximus script (should be placed after Parallaximus styles definition) -->
<script type="text/javascript" src="parallaximus/jquery.parallaximus.min.js"></script>

3b. For MooTools version (click for jQuery) include this scripts:

<!-- Parallaximus requires MooTools 1.4+ -->
<!-- Note: if you already have MooTools at this moment, you shouldn't include it once more. -->
<script type="text/javascript" src="parallaximus/mootools-core-1.4.5-full-nocompat-yc.js"></script>

<!-- Parallaximus script (should be placed after Parallaximus styles definition) -->
<script type="text/javascript" src="parallaximus/mootools.parallaximus.min.js"></script>

4. Add the parallaximus HTML code to the body section. The easiest way to do it is to copy one of attached demos html.
Layers should be placed as the child elements of main container, all original classes should be preserved.
Each layer can contain several images.
Note: you need to fill images' widths and heights.

<div class="w-parallaximus width_fixed height_fixed">
	<div class="w-parallaximus-layer index_1">
		<img src="image.png" width="165" height="219" alt="">
	</div>
	<div class="w-parallaximus-layer index_2">
		<img src="image.png" width="130" height="42" alt="">
	</div>
	<div class="w-parallaximus-layer index_3">
		<img src="image.png" width="182" height="28" alt="">
	</div>
</div>

5a. For jQuery version (click for MooTools) initialize the widget (usually at the end of the body section).
For settings list, check options section

<script>
jQuery(document).ready(function($){
	$('.w-parallaximus').parallaximus({
		// options are located here
		// for example: set return-effect duration to 5 seconds
		duration: 5000
	});
});
</script>

5b. For MooTools version (click for jQuery) initialize the widget (usually at the end of the body section).
For settings list, check options section

<script>
window.addEvent('domready', function(){
	new Parallaximus(document.getElement('.w-parallaximus'), {
		// options are located here
		// for example: set return-effect duration to 5 seconds
		duration: 5000
	});
});
</script>

6. Apply sizes to widget and layers using CSS.

To widget:

.w-parallaximus {
	width: 240px;
	height: 400px;
	/* If you widget has background place it here */
	background: url(background.jpg);
	}

To each layer:

(Note that layers should be bigger than widget itself. How to choose layer size? The main rule: the closer the layer to us, the bigger it should be).

.w-parallaximus-layer.index_1 {
	width: 400px;
	height: 500px;
	}
	.w-parallaximus-layer.index_1 > img {
		left: 20px;
		top: 30px;
		}

7. Choose the way you widget resizes by adding or removing container classes width_fixed and height_fixed

Fixed width and height (example):

<div class="w-parallaximus width_fixed height_fixed">

Responsive width with fixed height (example):

<div class="w-parallaximus height_fixed">

Responsive width and height (example):

<div class="w-parallaximus">
